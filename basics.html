Fundamentals

There are several tags that are neccesary for an HTML document to function properly. Every webpage must have a document decleration: <!DOCTYPE html>. This lets the browser know that the file it is trying to display is an HTML document. The decleration does not need a closing tag, like most other tags do. The HTML document begins after the decleration. It consists of an opening and closing html tag: <html> </html>. All the content you wish to display goes in between the html tags. A set of tags with information within them are known as an element. Every HTML element can have different attributes. Attributes give extra information about the element and usually consist of a name and value. The name describes what part of the element should be changed, such as color, size or border. The value gives information such as the name of the color and amount of pixels.

The Head

The first set of tags that should go within your html tags should be the head tag: <head> </head>. The head tag contains information about the page that the user will not see, but still impacts how it looks. The head will hold all of the metadata about the HTML document. Metadata is the information about the page that the web browser can understand. It can affect how the website looks and functions. The head is also where any extrnal sources like a CSS or Google Font goes. The last important tag that goes in the head is the <title> tag. The title tag tells the browser what text should be displayed in tabs. One example is having homepage as a tab so the user can more easily navigate your site.

The Body

This where most of the content on a web page will go. That includes all text, images and other media types that the user is intended to see. Everything within <body> </body> will be displayed by the browser. The headings tags cotain text that will be enlarged and are used for headlines and titles. Headings have six different tags each indicating a specific size. They range from <h1> to <h6> and go from biggest to smallest. Most text and content that will be seen will go within paragraph tags: <p>. Some other commonly used elements within the body are links and images.

Links & Images

One of the most important aspects of HTML is linking websites and webpages together. A link is displayed using the <a> tag and needs to have an attribute directing to the link. The attribute is defined by using href="address" in the starting tag. It also needs a name which can be either characters or an image. Links & images can both be used from a local file or from the internet. For example, if your website has multiple pages you can link them together or link to another source. Images use the <img> tag and are one of the elements that does not need to be closed. In order for an image to display there has to be the img tag as well of the source of the picture. Example: <img src="imageaddress">

Images can be resized using the width & height attributes. It is also advisable to use an alt which give the image a title that shows up if the image does not show and helps visually impaired users.
Example:<img src="imageaddress" alt="image" width="50px" height="50px">

Link with characters example: <a href="https://www.google.com">Google</a> will display as Google in a browser.
