Attribute

Additional information on tag that changes certain aspects about it, such as size and color.

Body

The container that holds all of the pages contents, such as images, text and links.

CSS

Stands for cascading style sheet. External file that is used to change the apperance of a webpage.

Head

Tag that goes at the beggining of the HTML document. Users cannot see it, but holds information for the browser.

Headings

Tags that go around certain text that will be used as titles on a page. Has six sizes.

HTML

Hyper Text Markup Language. The language used to create websites. Can only be interpretted by web browsers.

Images

Img tag is used to display pictures on webpage. Needs a src address to the image. Does not need closing tag.

Links

Connect documents to other sources such as other websites or webpages of the same site.

Paragraphs

One of the most common HTML tags. Is used to hold text and other elements inside.

Tags

Divide parts of HTML into elements. They are made up of specific words inside of angled brackets. A majority of elements need an opening and closing brackets, however there are some exceptions.

Title

Tag that tells the browser what to display in the tab of your website. Helps the user navigate.
